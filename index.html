Benchmark Project Page

ADAM-Bench: Auditing Scientific Dialogue with Paper-Grounded Evidence

ADAM-Bench evaluates whether scientific dialogue claims are supported by evidence in the corresponding paper PDF. The public release includes layered benchmark data, annotation system code, leaderboard code, and key data processing scripts.

Overview

Public Benchmark Components

Layered Data Release

3

Scale, Core, and Gold layers for staged research usage.

Download Archives

3

Layer-level tarballs with SHA256 checksum files.

Benchmark Tasks

2

Task A: label prediction. Task B: minimal evidence localization.

Evaluation Metrics

3

Macro-F1, Evidence-F1, and FEVER-style end-to-end score.

Task A

4-way paper-grounded claim classification: SUPPORTED, CONTRADICTED, NOT_FOUND, UNDECIDABLE.

Task B

Minimal evidence set localization over unified evidence objects (eobj_id).

Protocol

Gold uses set-of-sets evidence supervision and best-match scoring for robust evaluation.

Run Evaluator (Code-Only Example)

PYTHONPATH=src python3 -m reviewground.eval.evaluate \
  --gold /path/to/gold.dev.jsonl \
  --pred /path/to/predictions.jsonl

Launch Leaderboard Service

PYTHONPATH=src python3 leaderboard/app/server.py \
  --config configs/build_core.yaml \
  --models configs/models.yaml \
  --runs-dir leaderboard/runs \
  --host 127.0.0.1 --port 9000
